| <h2>**問題1：為什麼需要住宅代理？**</h2>                                                                |
| :----------------------------------------------------------------------------------------- |
| 無論是用機器人程式或各類工具進行網頁數據採集，還是在網站上搶購商品，住宅代理都是必備工具。借助住宅代理，你能同時運行更多任務，降低帳號被封禁的概率，並提升訪問大多數網站的連接速度。 |

| <h2>**問題2：住宅代理與 ISP 代理有什麼區別？**</h2>                                                                                                                                    |
| :--------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| ISP 代理為固定專屬類型，不支持 IP 輪換，配備獨立 IP 池且不限流量；而住宅代理基於流量計費，使用共享 IP 池，且提供輪換和粘性的IP會話類型：輪換 IP 模式下，每次請求 / 刷新都會更換 IP；粘性IP 模式下，只要會話處於活躍狀態，IP 就會保持穩定，可持續時間為 5-180 分鐘，直至 IP 自然下線後才會變更。 |

| <h2>**問題3：我可以指定代理的目標城市嗎？**</h2>             |
| :------------------------------------------ |
| 可以的。使用住宅代理可精准指定目標地理位置，無論是具體城市、州還是國家，均可按需選擇。 |

| <h2>**問題4：SOCKS5 和 HTTP 代理，應該如何選擇？**</h2>                                                                                     |
| :---------------------------------------------------------------------------------------------------------------------------- |
| 與僅能處理 HTTP 和 HTTPS 網頁流量的 HTTP 代理不同，SOCKS5 代理可相容任何類型的網路流量。但 SOCKS5 代理存在局限性，無法直接在 Google Chrome、Brave 及 Microsoft Edge 等流覽器中使用。 |
