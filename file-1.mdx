## 一、使用前準備

### 1. 訪問定價頁面下載客戶端

下載並安裝 LokiProxy 客戶端後，即可立即體驗高性價比的短效住宅 IP 服務，快速完成代理配置與管理。

### 2. 檢查本地網絡環境

在打開 LokiProxy 客戶端後，您可以先查看 **本機 IP**：

- 若顯示為 **非中國大陸 IP**，則說明當前網絡環境可以正常使用客戶端
- 若為中國大陸 IP，則需要先調整網絡環境

### 3. 計費說明

- **按 IP 計費**：每提取並使用一個 IP，消耗 1 個 IP 額度
- **24 小時內復用不扣費**：同一 IP 在 24 小時內可反覆使用
- **套餐內 IP 額度無有效期限制**：未使用不過期

## 二、開始使用

### 1. IP 庫

IP 庫中展示可用的住宅 IP 資源，支持以下操作：

#### （1）轉發端口

- 選擇某一個 IP
- 右鍵選擇一個端口並完成綁定
- 綁定成功後，可在【端口轉發】中查看並使用

> **扣費規則**：每綁定一次端口，即消耗 1 個 IP 額度

#### （2）批量轉發

- 點擊【批量轉發】
- 在彈窗中輸入需要轉發的數量（單次最多 100 個）
- 點擊【開始批量轉發】，系統將依序完成端口綁定

> 當批量轉發數量大於剩餘 IP 額度時，將無法執行該操作

#### （3）代理到進程

- 選擇某個 IP
- 右鍵將代理指定到某個程序
- 前提是已在【程序代理】中添加對應程序（如 Chrome、Firefox 等）

---

### 2. 24 小時列表

- 成功綁定後的 IP，會顯示在【24 小時列表】中
- **24 小時內重複使用不會再次扣費**
- 超過 24 小時後，IP 將自動從列表中移除

---

### 3. 端口轉發

- 所有已綁定端口的 IP 都會顯示在此列表
- 點擊某一 IP，即可複製 **本地地址 + 端口**
- 可直接粘貼到第三方代理工具中使用

---

### 4. 程序代理

程序代理可讓代理 IP 直接作用於指定應用，無需手動配置。

#### （1）智能導入

- 點擊【智能導入】
- 客戶端會自動掃描電腦中的瀏覽器並添加

#### （2）添加程序

- 點擊【添加程序】
- 在本地選擇對應程序的 `.exe` 文件即可完成添加

#### （3）使用程序代理

- 點擊已添加的程序，選擇【啟動應用】
- 在彈窗中選擇代理 IP（可來自 IP 庫 / 24 小時列表 / 端口轉發）

> 若從 IP 庫直接選擇 IP，將立即扣除 1 個 IP 額度
