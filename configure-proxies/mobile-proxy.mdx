Before configuring the mobile proxy, please ensure that you have added an IP whitelist or a sub-user.

## **Accessing Proxies via API**

1. Click "PROXY" and select "Mobile Proxy"

2. Select "API (IP Whitelist)" and click "Proxy Setup"

3. Select the desired country/region or opt for“Global Mix”

4. Decide whether to enable the ASN configuration feature based on your actual needs

5. Enter the number of IPs to extract (maximum 900)

6. Select "Protocol Type", "Format", "Separator", and "Session Type" as needed

7. Select the “hostname“. To ensure the best connection, choose the nearest server based on your network IP address

8. Click "Generate API URL" to display it in the "Get API URL" section on the right

9. Click "Open URL" to view the available proxy list.

## **Accessing Proxies via Username/Password**

1. Click "PROXY" and select "Mobile Proxy"

2. Select "Username:Password" and click "Proxy Setup"

3. Select the desired country/region or opt for“Global Mix”

4. Decide whether to enable the ASN configuration feature based on your actual needs

5. Select the target city or choose Random.

6. Select "Session Type". Choose "Rotating IP" to rotate the IP with each request, or choose "Sticky IP" to set a fixed IP duration.

7. Select the hostname. To ensure the best connection, choose the nearest server based on your network IP address.

8. Select the "Sub-user" that needs to access the proxy.

9. Finally, select the number of proxies and format on the right side of the dashboard, then click "Generate URL" to obtain the available proxy list in the "Proxy Generator".
