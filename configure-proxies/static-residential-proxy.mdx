Static residential proxies support username/password authentication. Before using a static residential proxy, please ensure that you have purchased the corresponding plan.

Note: The IPs displayed in the static IP list require no manual configuration. Simply copy the generated proxy parameters and they can be used in any program or tool.

① Click "PROXY" and select "Static Residential".

<Frame>
  ![英文1](/images/英文1.png)
</Frame>

② Based on your actual needs, select the IP address you require and copy its "Proxy Command".

③ Paste it into your application or tool to complete the configuration.

④ To configure the IP on a mobile device, click "QR Code" \> "View", then open the corresponding APP and scan the code to start using it.
