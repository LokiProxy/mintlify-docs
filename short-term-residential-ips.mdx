## **Q1：How do I use the Short-Term Residential IP plan?**

To use the short-term residential IP plan, you can download and install our client app and obtain IPs through port forwarding. Alternatively, log in to the LokiProxy dashboard and access the service via API or authentication.

## **Q2：Where can I get the IP list?**

You can access the full IP list via the LokiProxy dashboard or by downloading the LokiProxy client app. You can filter proxies by country, state, or city to precisely locate the IPs you need.

## **Q3：How long can an IP stay active?**

For residential IPs, the duration varies by IP. From the moment you start using it until the IP naturally goes offline, the connection remains stable. If an IP goes offline and later comes back online, you can reuse it for free within the 24-hour list.

## **Q4：Can I reuse extracted proxies?**

Yes. Any proxy you have extracted can be reused for free within 24 hours via the “24-hour list” in the dashboard.

## **Q5：Do unused IP credits expire?**

No. Unused IP credits never expire. Your balance is only deducted when you extract an IP for the first time.
