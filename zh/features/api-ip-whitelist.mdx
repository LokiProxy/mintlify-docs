您可以通過API（IP白名單）或用戶名/密碼訪問代理服務，其中API（IP白名單）支持動態住宅代理和不限量住宅代理。若要通過API訪問代理，請先添加白名單IP，接下來將爲您說明配置和管理方法。

## 新增白名單IP

1. 點擊儀表盤側邊欄需要使用的代理服務，以動態住宅代理爲例。
2. 點擊“API（IP白名單）”\>“管理IP白名單”
3. 點擊“新增白名單”，在“白名單IP”欄中填入需要使用代理的網絡環境對應的IP地址
4. 狀態選擇“開啓”\>”提交“，配置完成

如果您需要立即開始使用，請參閱”[配置代理](/zh/configure-proxies/rotating-residential-proxy)“部分的快速指南。

## 管理IP白名單

點擊“管理IP白名單”，在該頁面可進行管理：

1. 搜索IP：在”輸入IP地址“欄中輸入完整IP地址\>”搜索“
2. 備註：點擊白名單IP後方編輯圖標，填寫後提交
3. 批量備註：勾選任意個需要操作的白名單IP，點擊“批量備註”\>輸入備註信息\>”提交“
4. 刪除：在需要操作的白名單IP後方點擊垃圾桶圖標
5. 批量刪除：勾選任意個需要操作的白名單IP，點擊“批量刪除”\>”確認“

如果您需要立即開始使用，請參閱”[配置代理](/zh/configure-proxies/rotating-residential-proxy)“部分的快速指南。
