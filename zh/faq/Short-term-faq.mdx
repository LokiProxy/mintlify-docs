---
title: "短效住宅IP"
---

| <h2>**問題1：我如何使用短效住宅 IP 套餐？**</h2>                                               |
| :------------------------------------------------------------------------------ |
| 要使用短效住宅 IP 套餐，可以下載安裝我們提供的客戶端應用，通過端口轉發獲取IP；或者登錄 LokiProxy 儀表盤，通過 API 或者身份驗證即可使用。 |

| <h2>**問題2：我在哪裏可以獲取 IP 列表？**</h2>                                                    |
| :---------------------------------------------------------------------------------- |
| 您可以通過 LokiProxy 的儀表盤或下載LokiProxy的客戶端應用，來訪問完整的 IP 列表。您可以按國家、州/省或城市篩選代理，以準確找到您所需的 IP。 |

| <h2>**問題3：IP 可以保持活動狀態多長時間？**</h2>                                                                 |
| :------------------------------------------------------------------------------------------------ |
| 對於住宅 IP 而言，每個 IP 的具體持續時間 都是不同的，從開始使用到 IP 自然下線，均可保持穩定的連接。如果某個 IP 離線後再次變爲在線狀態，您可以在 24 小時列表內免費重複使用它。 |

| <h2>**問題4：我可以重複使用已提取的代理嗎？**</h2>           |
| :----------------------------------------- |
| 是的。您已提取的代理可以在 24 小時內，通過儀表盤的“24小時列表”免費重複使用。 |

| <h2>**問題5：未使用的 IP 餘額會過期嗎？**</h2>        |
| :-------------------------------------- |
| 不會，未使用的 IP 餘額永不過期。您的餘額僅在您首次提取 IP 時才會扣費。 |

| <h2>**問題6：Lokiproxy的短效住宅IP高級套餐有哪些優勢？**</h2>                      |
| :--------------------------------------------------------------- |
| CDKey靈活分配：便於團隊內部分配使用，權限管理更清晰； 支持自由轉售：高級套餐用戶享有IP資源轉售權，可將閒置額度靈活變現。 |

| <h2>**問題7：如何使用短效住宅IP高級套餐？**</h2>                                            |
| :-------------------------------------------------------------------------- |
| 您可以根據需求和用量購買對應級別的套餐，然後訪問儀錶板-短效住宅IP-CDKey，根據要使用的IP數量生成CDKEY，然後在設置代理頁面兌換即可使用。 |
