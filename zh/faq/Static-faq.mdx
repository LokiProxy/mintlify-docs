title: "靜態住宅代理"
---

| <h2>**問題1：什麼是靜態住宅 ISP 代理？**</h2>                                 |
| :--------------------------------------------------------------- |
| 靜態住宅 ISP 代理是由互聯網服務提供商分配的真實家庭 IP，這些 IP 地址會在較長時間內保持固定不變（通常為數月或更久）。 |

| <h2>**問題2：靜態 ISP 代理會輪換 IP 嗎？**</h2>                       |
| :-------------------------------------------------------- |
| 通常不會。除非被互聯網服務提供商收回，否則 IP 地址會保持固定。部分服務商可根據需求提供手動更換 IP 的服務。 |

| <h2>**問題3：靜態住宅代理支持SOCKS5協議嗎？**</h2>         |
| :------------------------------------------ |
| 我們的靜態住宅代理均支持HTTP和SOCKS5協議，並通過用戶名+密碼的方式進行認證。 |

| <h2>**問題4：LokiProxy靜態住宅代理有流量限制嗎？**</h2>               |
| :---------------------------------------------------- |
| LokiProxy的靜態住宅代理按時間週期計費，流量使用無上限。提供穩定帶寬，可滿足長期項目持續運行需求。 |

| <h2>**問題5：為什麼推薦使用LokiProxy的靜態住宅代理？**</h2>                                |
| :----------------------------------------------------------------------- |
| LokiProxy 的靜態住宅代理的原生住宅IP資源，是來自真實互聯網服務提供商的穩定且獨享的 IP 地址，可確保您的業務長期穩定的連接和性能。 |

| <h2>**問題6：LokiProxy的靜態住宅代理支持續期使用嗎？**</h2>                       |
| :-------------------------------------------------------------- |
| 是的。LokiProxy的靜態住宅型代理支持續費功能，您只需在IP到期前延長有效期，就可以使用同一獨享 IP 來繼續您的業務。 |
