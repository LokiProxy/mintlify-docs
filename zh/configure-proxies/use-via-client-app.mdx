## 一、使用前須知

### 1. 下載並安裝

通過[定價頁](https://www.lokiproxy.com/zh-TW/pricing/short-term-residential-ip)或[儀錶盤](https://dashboard.lokiproxy.com/zh-TW/)下載並安裝 LokiProxy 客戶端後，使用已註冊的郵箱登錄，如您是通過谷歌郵箱一鍵登錄，需要先在儀表盤設置密碼。

### 2. 檢查本地網絡環境

在打開 LokiProxy 客戶端後，您可以先查看 **本機 IP**：

- 若顯示為 **非中國大陸 IP**，則說明當前網絡環境可以正常使用客戶端
- 若為中國大陸 IP，則需要先調整網絡環境

### 3. 計費說明

- **按 IP 計費**：每提取一個 IP，消耗 1 個 IP 額度
- **24 小時內復用不扣費**：同一 IP 在 24 小時內可反覆使用
- **套餐內 IP 額度**：未使用不過期

## 二、開始使用

### 1. 在 IP 庫中選擇所需 IP

IP 庫中默認展示 **100** 條可用的住宅 IP 資源，下滑加載後，會多顯示100條數據。點擊**刷新**按鈕可刷新當前IP庫資源。支持通過以下3种操作獲取IP：

#### （1）轉發端口

- 右鍵點擊某一個 IP，選擇一個可用端口並完成綁定
- 綁定成功後，可在【端口轉發】中查看並使用

<Note>
  IP 轉發到端口後，即消耗 1 個 IP 額度，後續24小時內不再重復扣費
</Note>

#### （2）批量轉發

- 點擊【批量轉發】
- 在彈窗中輸入需要轉發的數量（單次最多 100 個）
- 點擊【開始批量轉發】，系統將依序完成端口綁定

<Note>
  當批量轉發數量大於剩餘 IP 額度時，將無法執行該操作
</Note>

#### （3）代理到進程

- 選擇某個 IP
- 右鍵將代理指定到某個程序

<Note>
  前提是已在【程序代理】中添加對應程序（如 Chrome、Firefox 等）
</Note>

### 2. 查看已提取IP

#### （1）通過【24小時列表】查看

成功綁定後的 IP，會顯示在【24 小時列表】中。24 小時內，IP狀態為【**在線**】可以重複使用，不會再次扣費。超過 24 小時後，IP 將自動從列表中移除。

#### （2）通過【端口轉發】查看

所有已綁定端口的 IP 都會顯示在此列表。點擊某一 IP或者右鍵，即可複製 \*\*本地地址 + 端口，\*\*可直接粘貼到第三方代理工具中使用。

<Note>
  關閉客戶端後，已綁定的端口會**全部釋放**，需要在【24小時列表】中重新轉發到端口，才能繼續使用。
</Note>

## 三. 其他功能

### 1. 程序代理

可讓代理 IP 直接作用於指定應用，無需手動配置。

#### （1）智能導入

- 點擊【智能導入】
- 客戶端會自動掃描電腦中的瀏覽器並添加

#### （2）添加程序

- 點擊【添加程序】
- 在本地選擇對應程序的 `.exe` 文件即可完成添加

#### （3）使用程序代理

- 點擊已添加的程序，選擇【啟動應用】
- 在彈窗中選擇代理 IP（可來自 IP 庫 / 24 小時列表 / 端口轉發）

<Note>
  若從 IP 庫直接選擇 IP，將立即扣除 1 個 IP 額度
</Note>

### 2. 設置

端口取值範圍為：1025~65535，您可在此區間內自定義起始端口值，和端口數量。
