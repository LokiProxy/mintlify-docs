---
title: "如何使用靜態住宅代理？ "
sidebarTitle: "靜態住宅代理"
---

靜態住宅代理支持用戶名/密碼的驗證方式。在使用靜態住宅代理之前請先確認您已經購買相應套餐。

**溫馨提示：**

<Info>
  您購買的靜態IP會顯示在您的可用額度中，未分配激活前，該 IP 額度永久有效；
</Info>

<Info>
   IP 提取分配成功後，您的可用額度會相應的減少。
</Info>

<Info>
  已提取的 IP 可以在靜態 IP 列表中查看。
</Info>

**使用步驟：**

① 進入儀錶盤 -「靜態住宅代理」

② 在「提取IP」界面，查看您的「可用額度」，這裡顯示您當前剩餘可提取的 IP 數量。（如果可用額度為0，請先購買所需的靜態地區和數量）

<Frame>
  ![1 3](/images/1-3.png)
</Frame>

③ 首先在「選擇地區」步驟勾選您要提取的國家/地區，或選擇全部提取。

<Frame>
  ![2 5](/images/2-5.png)
</Frame>

④ 然後選擇您偏好的 IP 網段，或由系統隨機分配，選擇對應的 IP 數量，點擊進入下一步「確認分配」。

<Note>
  注意：如果您指定的 IP 網段庫存數量＜您的可用額度，則最大只能提取庫存數量。
</Note>

<Frame>
  ![3 3](/images/3-3.png)
</Frame>

⑤ 核對您的提取信息，確認信息無誤後點擊\[立即激活\]。

<Note>
  **本次共計消耗**指的是您本次提取消耗的可用 IP 額度。
</Note>

<Note>
  **提取目標地區**指的是您在「選擇地區」步驟中選擇的地區數。
</Note>

<Frame>
  ![4 2](/images/4-2.png)
</Frame>

⑥ IP 提取後需要等待 1-2 分鐘。提取成功則扣減相應的可用額度，您可以在 \[靜態IP列表\] 中查看已提取的IP地址，並複製其「代理指令」。將複製好的「代理指令」粘貼至應用程式或工具中，即可完成配置。

<Frame>
  ![5 3](/images/5-3.png)
</Frame>

⑦ 若需將 IP 配置至手機設備，請點擊「二維碼」\>「查看」，並開啟相應APP掃碼即可使用。

<Frame>
  ![6 1](/images/6-1.png)
</Frame>
