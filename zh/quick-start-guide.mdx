快速入門LokiProxy，瞭解如何進行身份驗證以及使用您購買的代理服務。

LokiProxy提供API ( IP 白名單) 和用戶名/密碼的身份驗證方式幫助用戶配置代理。API ( IP 白名單) 僅支持動態住宅代理，不限量代理，用戶名/密碼支持所有代理類型。

若您想要先熟悉關於身份驗證方式的說明，以及對應的代理配置方法，請參閱相應的入門指南：

- [新增白名單IP](/zh/features/api-ip-whitelist)
- [新增子用戶](/zh/features/username-password)
- [動態住宅代理](/zh/configure-proxies/rotating-residential-proxy)
- [不限量住宅代理](/zh/configure-proxies/unlimited-residential-proxy)
- [靜態住宅代理](/zh/configure-proxies/static-residential-proxy)
