LokiProxy supports two authentication methods for proxy configuration: **API (IP Allowlist)** and **Username/Password**. The API (IP Allowlist) method is only compatible with Rotating Residential Proxies and Unlimited Proxies, while the Username/Password method supports all proxy types.

If you'd like to familiarize yourself with these authentication methods and their corresponding setup procedures, please refer to the relevant beginner guides:

- **Adding an IP to the Allowlist**
- **Creating a Sub-account**
- **Rotating Residential Proxies**
- **Unlimited Residential Proxies**
- **Static Residential Proxies**
