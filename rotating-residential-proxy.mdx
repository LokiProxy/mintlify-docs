## **Q4：Why do l need residential proxies?**

Proxies are a necessity for any bot or tool when scraping or trying to purchase something from a website. Proxies allow you to run more tasks,decrease the odds of being banned, and have a faster connection to most websites.

## **Q5:What are the differences between Residential proxies and ISP proxies?**

ISP proxies are not rotating,they are dedicated with unlimited bandwidth and use private IP pool, while residential proxies based on bandwidthuse shared IP pool and have 2 options that can be chosen anytime you want: Random IP rotates every request/refresh, sticky IP lets your IP staystatic as long as the session is active, at least for 5-180 minutes, and then until the server restarts.

## **Q6：Can lchoose which City my proxy will target?**

Yes, with residential proxies, you can specify the geographic location you want to target, whether that's a specific city, state, or country.

## **Q7：SOCKS5 VS HTTP, which should I choose?**

Unlike HTTP proxies, which can only interpret and work with HTTP and HTTPS webpages, SOCKS5 proxies can work with any traffic. But SOCKS5come with limitations of not being able to be used on browsers such as Google Chrome, Brave and Microsoft Edge.
